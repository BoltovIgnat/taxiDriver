# Промпты Lovart — генерация изображений

Промпты для создания **фото и иллюстраций** под каждую страницу сайта набора водителей такси.

## Как использовать

1. Открой файл нужной страницы (`01-home.md` … `20-404.md`).
2. Скопируй **один блок** `IMG-N` целиком (стиль + промпт + negative уже внутри).
3. Вставь в Lovart, укажи **Ratio** из заголовка блока.
4. Общие правки стиля — `_STYLE-BASE.md`, негатива — `_NEGATIVE.md` (источники правды).

## Рекомендуемые размеры

| Тип | Ratio | Размер (px) | Где используется |
|-----|-------|-------------|------------------|
| Hero desktop | 16:9 | 1920×1080 | Hero справа / фон |
| Hero mobile | 4:5 | 800×1000 | Hero mobile |
| Card thumbnail | 4:3 | 800×600 | Карточки городов, блог |
| Square avatar | 1:1 | 400×400 | Отзывы |
| OG / share | 1.91:1 | 1200×630 | meta og:image |
| Icon illustration | 1:1 | 512×512 | Шаги, FAQ, CTA |

## Workflow

1. **01-home IMG-1 (hero)** → утверди визуальный стиль.
2. **03-city-page** → городские фоны по шаблону.
3. Остальные — пакетами по страницам.

## Связь с вёрсткой

Промпты для HTML/CSS лежат в `prompts/lovart/`.  
Промпты для картинок — здесь, в `prompts/lovart/images/`.
