# Политика конфиденциальности `/privacy`

## Базовые требования (HTML/CSS)

Сверстай landing page для сайта набора водителей такси в России.

## Формат выхода (обязательно)

- Только чистый **HTML5 + CSS** (один файл `index.html` со встроенным `<style>` или отдельный `style.css`).
- Без React, Vue, Bootstrap, Tailwind, jQuery.
- Семантическая разметка: `header`, `nav`, `main`, `section`, `footer`.
- Адаптив: mobile-first, breakpoints **360px / 768px / 1440px**.
- Все тексты на русском, **без Lorem Ipsum** — используй реалистичный контент из промпта страницы.
- CSS: custom properties для цветов, flexbox/grid, без тяжёлых анимаций.

## Дизайн-система

- **Стиль:** надёжный, «автомобильный сервис», вызывает доверие. Не стартап-neon, не glassmorphism.
- **Палитра:**
  ```css
  --bg: #F5F6F7;
  --surface: #FFFFFF;
  --text: #1A1D21;
  --muted: #6B7280;
  --steel: #4A5568;
  --accent: #2563EB;
  --border: #E5E7EB;
  ```
- **Шрифт:** system-ui или Inter (Google Fonts).
- **Container:** max-width 1200px, padding 16–32px.
- **Типографика:** H1 32–48px mobile / 48–56px desktop.
- **Секции:** padding 48–80px, много «воздуха».
- **Карточки:** белый фон, border 1px `#E5E7EB`, border-radius 16px, лёгкая тень.
- **Кнопка primary:** фон `#2563EB`, белый текст, radius 12px, padding 14px 24px.
- **Кнопка secondary:** белый фон, border `#D1D5DB`.

## Контент (max-width 720px, font-size 14px, line-height 1.6)

Вступление:

«Настоящая Политика определяет порядок обработки персональных данных пользователей сайта в соответствии с Федеральным законом № 152-ФЗ.»

### 1. Какие данные мы собираем

Имя, номер телефона, город, данные калькулятора дохода, URL страницы отправки формы.

### 2. Цели обработки

Обработка заявок на подключение водителей, обратная связь, передача данных таксопарку-партнёру.

### 3. Хранение данных

Данные хранятся на серверах на территории Российской Федерации. Срок хранения — до достижения целей обработки или отзыва согласия.

### 4. Права пользователя

Вы вправе запросить удаление или изменение персональных данных, направив запрос на контактный телефон или email, указанный на сайте.

## Стиль

Минималистичная текстовая страница. Без формы, без CTA. Header + footer только.

Используй тот же header и footer, что на главной.
